# Progressly Enterprise Admin Panel Overview

This document details all 16 views supported by the Progressly Enterprise Admin Panel:

1. **Dashboard**: Summarized stats, charts, database table sizes.
2. **Users**: List users, edit roles, password resets.
3. **Skills**: Skill sets progress metrics.
4. **Tasks**: Status updating and view switcher (Kanban, Grid, Table, Timeline).
5. **Streaks**: Activity heatmap calendars.
6. **Feedback**: Moderating tickets.
7. **Email Logs**: Delivery details and sending retries.
8. **Notifications**:Broadcast dispatches.
9. **Analytics**: Recharts graphs.
10. **Reports**: CSV downloads.
11. **Database**: PostgreSQL index logs.
12. **System Health**: Port status diagnostic checks.
13. **Backup**: Database snapshots creation/restoration.
14. **Settings**: Configuration updates.
15. **Activity Logs**: Security audits history.
16. **Roles**: Group matrix mappings.
